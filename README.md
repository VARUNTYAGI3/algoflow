# AlgoFlow

AlgoFlow is an interactive DSA visualization platform that helps students understand algorithms through step-by-step execution and animations.

## Features

- Binary Search Visualizer
- BFS Visualizer
- DFS Visualizer
- Merge Sort Visualizer
- Timeline Controls
- Execution Insights Panel
- Responsive Dashboard
- Real-time Algorithm Animation

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
